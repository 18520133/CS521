Hướng dẫn sử dụng:
Bước 1: Bỏ file requirements.txt ở folder ecc_visualize vào đúng đường dẫn : This PC->Windows(C:)->Users->Admin
Bước 2: Mở command promt chạy lệnh python3 -r requirements.txt để cài dặt các thư viện cần thiết
Bước 3: Vào lại folder acc_visualize nhấn Shift+chuột phải chọn Open PowerShell window here chạy lệnh python -u main.py
Bước 4: Mở Chorme vào trang http://127.0.0.1:8000/IR
